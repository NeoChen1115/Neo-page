## 🔍 特殊情况分析

### 0005.HK - 数据漂移问题

**问题根源**：严重的数据分布漂移

```
训练集范围: [20.96, 56.74]
测试集范围: [53.50, 110.80]  ❌ 超出训练范围

决策树的问题:
- 是"分段常数"模型，无法外推
- 在超出训练范围的数据上，无法做出有效预测
- 所有超大值都被映射到同一个叶子节点
- 结果：100%常数预测（56.36）
```

**为什么其他模型没这个问题**：
- **线性回归**：线性模型，自然可以外推
- **LSTM**：神经网络学到通用模式，能处理新范围的数据

**结论**：这是数据特性问题，而非模型bug
- 0005.HK 在测试期间有大幅上涨
- 导致决策树无法适应

### 0288.HK - 轻微改进的原因

```
训练集范围: [2.98, 6.31]
测试集范围: [4.14, 8.56]

虽然有data drift，但程度更轻：
- 范围扩大不如0005那么剧烈
- 超参数调优有帮助
- 但改进空间仍有限（5.5%）
```
